Detailed Description

Class for interfacing with shlibpp plugin library.

This helper class ease the integration of shlibpp within the BlockFactory framework. It is implemented with a singleton pattern.

It can handle multiple plugin libraries together and provides support of destructing the related factory objects (and hence unloading the plugin) when all the extracted classes have been destroyed.
